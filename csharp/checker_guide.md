# C# 学習計画 - AI チェックガイド

各 Station のクリア判定を **AI チェックプロンプト（Copilot Chat）** で行います。

---

## 使い方

1. 各 Station の **AI チェックプロンプト** をコピーする
2. VS Code の **Copilot Chat** を開く（`Ctrl+Shift+I`）
3. プロンプトを貼り付けて送信する
4. AI が **PASS / FAIL** と理由を返す

---

## AI チェックプロンプトの形式

各プロンプトは以下の構造です：

```
あなたは C# の学習チェッカーです。
以下のクリア条件に基づいて、学習者のコードを判定してください。

## クリア条件
- （条件1）
- （条件2）

## 判定ルール
- すべての条件を満たしていれば「✅ PASS」と表示
- 満たしていない条件があれば「❌ FAIL」と表示し、何が不足かを具体的に説明
- 改善のヒントも 1〜2 行で添える

## 対象コード
（← ここに自分のコードを貼り付ける、またはワークスペースのファイルを指定する）
```

### Tips

- **`@workspace`** を使うとプロジェクト全体を対象にチェックできます
  - 例: `@workspace Station 6 のクリア条件をチェックしてください`
- **ファイルを指定** して聞くこともできます
  - 例: `#file:Item.cs このファイルが Station 6 の条件を満たしているか判定して`
- AI チェックで **コード品質** の改善提案を受けることがあります（より良い書き方の提案など）

---

## 判定フロー

```
Station の問題を解く
    ↓
[AI チェック] プロンプトを Copilot Chat に貼り付け
    ↓ PASS?
次の Station へ！
```
